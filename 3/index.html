TP : créer une calculatrice

Objectif du tp créer une calculatrice capable d'effectuer les opérations suivantes

  • Opération mathématiques
    • Addition => "addition"
    • Soustraction => "soustraction"
    • Multiplication => "multiplication"
    • Division => "soustraction"(Une coquille s'est glissée içi, c'est bien division, le nom de la fonction)

Addition

var addition = function(parametre1, parametre2){
	alert(parametre1 + parametre2);
};
addition(1, 2);

Soustraction

var soustraction = function(parametre1, parametre2){
	alert(parametre1 - parametre2);
};
soustraction(1, 2);

Multiplication

var multiplication = function(parametre1, parametre2){
	alert(parametre1 * parametre2);
};
multiplication(1, 2);

Division

var division = function(parametre1, parametre2){
	alert(parametre1 / parametre2);
};
division(4, 2);
